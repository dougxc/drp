# 🇨🇭 Doctor P's 50th Birthday: Madrid Rock'n'Roll course Bike Tour Quiz

A celebratory, interactive web application created for Doctor P's 50th birthday. This app combines a virtual tour around the Madrid Rock'n'Roll course with a humorous quiz based on Spanish and Australian facts.

## 🚵 The Journey
Doctor P's goal is to complete the **Madrid Rock'n'Roll Half Marathon**, starting and ending in Paseo de la Castellana.

### 🎮 Game Mechanics
- **Movement:** Doctor P's marker (featuring his photo!) only moves forward when a question is answered correctly.
- **The Penalty:** If Doctor P gets **two Spanish questions wrong in a row**, he is sent back one location on the map (but never further than the start!).
- **Aussie Bonus:** Every 4th question is a "funny" Australian question. Answering these correctly grants a **+2 location bonus jump** with no penalty for wrong answers.

## 🚀 How to Play
The game is hosted live at: **[https://dougxc.github.io/drp/](https://dougxc.github.io/drp/)**

### Local Development
If you want to run the project locally:
1. Clone the repo
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
